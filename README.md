## Requirements
Since Lindalë is built on Laravel 5.3.*, there are a few system requirements:

- PHP >= 5.6.4
- OpenSSL PHP Extension
- PDO PHP Extension
- PDO compliant database (SQL, MySQL, PostgreSQL, SQLite)
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
